# Checklist

结束前检查：

## Trigger

- `name` 与目录名一致，且只使用小写字母、数字、连字符
- `description` 同时说明“能做什么”和“什么时候用”
- `description` 明确了“全球股市观察 + A 股深度分析”的真实覆盖面

## Main File

- `SKILL.md` 只保留范围判断、路由和核心规则
- 所有 reference 都从 `SKILL.md` 直接可达
- 没有 Windows 本地绝对路径
- 没有把脚本实现、字段长表或旧版大段代码塞回入口文件

## References

- 每份 reference 只回答一类问题
- 交叉跳转使用相对路径和正斜杠
- 如果提到脚本而仓库里还没有，对应文案写成“建议脚本”或“若不存在则新增”

## Behavior

- A 股深度查询能落到正确 reference
- 全球股市相关请求如果超出当前深度覆盖，会先声明边界再继续能做的部分
- 东财相关任务会显式遵守共享限流和串行请求约束
- 批量任务优先脚本化，而不是在对话里重复生成大循环代码
